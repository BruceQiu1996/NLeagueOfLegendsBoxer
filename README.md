# NPhoenix Project Introduction

## 项目介绍
> NPhoenix是一款基于.net6 + vs2022 开发的一款桌面应用程序,主要功能如下所示:

💡 	|秒选英雄|自动接受对局|国服数据排行榜

💡	查看英雄对位压制|查看英雄优势对线|5v5符文配置|修改段位|修改生涯背景

💡	发送匹马信息到聊天界面|查看召唤师战绩|查看召唤师绝活英雄|查看召唤师战绩详情

💡  大乱斗秒选自己喜欢的英雄|大乱斗符文推荐
